1 .sra-.fastq
fastq-dump --split-3 1.sra
2 fastq quality filter: Fastx-toolkit
http://hannonlab.cshl.edu/fastx_toolkit/
usage: fastq_quality_trimmer [-h] [-v] [-t N] [-l N] [-z] [-i INFILE] [-o OUTFILE]
Part of FASTX Toolkit 0.0.13 by A. Gordon ( gordon@cshl.edu)

[-h] = This helpful help screen.
[-t N] = Quality threshold - nucleotides with lower
quality will be trimmed (from the end of the sequence).
[-l N] = Minimum length - sequences shorter than this (after trimming)
will be discarded. Default = 0 = no minimum length.
[-z] = Compress output with GZIP.
[-i INFILE] = FASTQ input file. default is STDIN.
[-o OUTFILE] = FASTQ output file. default is STDOUT.
[-v] = Verbose - report number of sequences.
If [-o] is specified, report will be printed to STDOUT.
If [-o] is not specified (and output goes to STDOUT),
report will be printed to STDERR.
usage: fastq_quality_filter [-h] [-v] [-q N] [-p N] [-z] [-i INFILE] [-o OUTFILE]
Part of FASTX Toolkit 0.0.13 by A. Gordon ( gordon@cshl.edu)

[-h] = This helpful help screen.
[-q N] = Minimum quality score to keep.
[-p N] = Minimum percent of bases that must have [-q] quality.
[-z] = Compress output with GZIP.
[-i INFILE] = FASTA/Q input file. default is STDIN.
[-o OUTFILE] = FASTA/Q output file. default is STDOUT.
[-v] = Verbose - report number of sequences.
If [-o] is specified, report will be printed to STDOUT.
If [-o] is not specified (and output goes to STDOUT),
report will be printed to STDERR.
fastq_quality_trimmer -t 20 -l 40 -Q 33 -i ERR1_1.fastq -o 1_1_trimmer.fastq -v
fastq_quality_filter -q 20 -p 50 -Q 33 -i 1_1_trimmer.fastq -o 1_1_1.fastq -v
pairmate.pl
3 ref index: BWA
http://bio-bwa.sourceforge.net
paper: Fast and accurate long-read alignment with Burrows-Wheeler transform.pdf, Fast and accurate short read alignment with Burrows-Wheeler transform.pdf
bwa index -a bwtsw ref.fasta
4 aln (paired-end reads)
bwa aln -e 10 -t 4 ref.fasta 1_1_1.fastq -f 1_1_1.sai
bwa aln -e 10 -t 4 ref.fasta 1_1_2.fastq -f 1_1_2.sai
5 .sai-.sam
bwa sampe -a 700 ref.fasta 1_1_1.sai 1_1_2.sai 1_1_1.fastq 1_1_2.fastq -f 1.sam
6 delete unmapped reads: SAMtools
http://www.htslib.org
paper: The Sequence Alignment/Map format and SAMtools.pdf
publications: http://www.htslib.org/doc/
samtools view -SF 4 1.sam > 1.F.sam
7 create dictionary of ref: Picard-tools
http://broadinstitute.github.io/picard/
https://github.com/broadinstitute/picard
java -jar picard.jar CreateSequenceDictionary OUTPUT=ref.sam R=ref.fasta
8 unsorted_file.sam
cat ref.sam > unsorted_file.sam && cat 1.F.sam >> unsorted_file.sam
9 sort
java -jar picard.jar SortSam INPUT= unsorted_file.sam OUTPUT=1.sorted.sam SO=coordinate
10 .sam-.bam
samtools view -bS 1.sorted.sam -o 1.bam
11 addhead
java -jar picard.jar AddOrReplaceReadGroups I=1.bam O=1.addhead.bam ID=i_1ID LB=i_1ID PL=Illumina PU=i_1PU SM=i_1
12 merge
java -jar picard.jar MergeSamFiles INPUT=1.addhead.bam INPUT=2.addhead.bam OUTPUT=i_1.bam
13 deduplicate
java -jar picard.jar MarkDuplicates MAX_FILE_HANDLES_FOR_READ_ENDS_MAP=8000 INPUT=i_1.bam OUTPUT=i_1.dedup.bam METRICS_FILE=i_1.dedup.metrics
14 realign: GATK
paper: The Genome Analysis Toolkit-A MapReduce framework for analyzing next-generation DNA sequencing data.pdf
https://www.broadinstitute.org/gatk/
samtools index i_1.dedup.bam;
java -jar GenomeAnalysisTK.jar -R ref.fasta -T RealignerTargetCreator -o i_1.realn.intervals -I i_1.dedup.bam;
java -jar GenomeAnalysisTK.jar -R ref.fasta -T IndelRealigner -targetIntervals i_1.realn.intervals -o i_1.realn.bam
-I i_1.dedup.bam;
15 variant calling
java -jar GenomeAnalysisTK.jar -glm SNP -l INFO -R ref.fasta -T UnifiedGenotyper -I i_1.dedup.bam -o i_1.vcf
--output_mode EMIT_ALL_SITES -stand_emit_conf 10 -stand_call_conf 50

paper: The variant call format and VCFtools.pdf
vcftools --vcf i_1.vcf --out SNPs_only --remove-indels --minQ 30 --thin 10 --min-alleles 2 --max-alleles 2
--kept-sites ;
